Sobre este trabalho

Este site foi desenvolvido para atender todos os requisitos da Atividade Somativa 1, usando exclusivamente HTML, CSS e JavaScript (sem frameworks).

O que o site contém

  • Menu fixo no topo (mesmo menu em todas as páginas).
  • Página com formulário que valida todos os campos.
  • Página que recebe dados via GET e os trata com JavaScript.
  • Arquivos organizados em pastas: /css, /js, /img.
