Sobre este trabalho

Este site foi desenvolvido para atender todos os requisitos da Atividade Somativa 1, usando exclusivamente HTML, CSS e JavaScript (sem frameworks).

O que o site contém:

  • Menu fixo no topo (padrão em todas as páginas).
  • Formulário validado via HTML e JS.
  • Página de recepção dos dados enviados por GET e tratada com JavaScript.
  • Arquivos organizados em pastas: css/, js/, img/.
